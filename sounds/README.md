# Звуки для Pomodoro Timer

Все звуки взяты из открытых источников и оптимизированы для веб-использования.

## Список звуков:

1. `bell.mp3` - Мягкий колокольчик
2. `chime.mp3` - Перезвон
3. `ding.mp3` - Короткий дзинь
4. `success.mp3` - Звук успешного завершения

## Использование

Звуки автоматически загружаются при старте приложения и могут быть выбраны в настройках.

## Источники

- Звуки взяты из библиотеки Mixkit (https://mixkit.co/free-sound-effects/)
- Оптимизированы для веб-использования через FFmpeg
